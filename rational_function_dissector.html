Rational Function Dissector

Factor completely to identify holes and vertical asymptotes

Step 1: Factor Completely
Original Function:
r(x) = (x² - 4)/(x² - x - 6)
Factored Form:
r(x) = (x - 2)(x + 2)/[(x - 3)(x + 2)]
Step 2: Identify Common Factors
Common Factors (Cancel):
Common factor: (x + 2)
Simplified Form:
r(x) = (x - 2)/(x - 3), x ≠ -2
Step 3: Classify Discontinuities
Summary
Key Rule: Common factors (appear in BOTH numerator and denominator) create HOLES. Factors only in denominator (after canceling) create VERTICAL ASYMPTOTES.
